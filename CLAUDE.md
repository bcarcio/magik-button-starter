# Building a Local Magik button

The agent instructions for this folder are tool-neutral and live in **[AGENTS.md](./AGENTS.md)**,
with the full `button.json` field reference in **[FORMAT.md](./FORMAT.md)**.

**Read both**, then help the user author `button.json` and run the loop:
`npm run validate` → `npm run dry-run` → `npm run publish`. A button is config (one
`button.json`), not code.
